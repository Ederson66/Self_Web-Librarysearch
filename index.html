1 sql server Xem SQL là viết tắt của Structured Query Language là ngôn ngữ truy vấn dữ liệu mang tính cấu trúc. SQL là loại ngôn ngữ máy tính phổ biến để tạo, sửa và lấy dữ liệu từ một hệ quản trị cơ sở dữ liệu quan hệ như: MySQL, Oracle, Database, MySQL Server. ...

STT Lệnh Ý nghĩa Design Tool
1 select * from sys.sysservers show phần mềm sql đag đc cài đặt trên con server nào
2 select * from sys.databases show tất cả các database đag nằm trên server của cta
3 select DB_NAME() AS 'Database Name' để xem cta đag nằm trên con database nào
4 Use tên database Dùng để sử dụng 1 con database khác
5 select * from sys.database_files xem lưu dữ liệu và file log đag được lưu trữ ở đâu
6 select * from sys.tables thể hiện tất cả các bảng trong database đó
7 select * from sys.objects where type='PK' khóa chính
8 select * from sys.objects where type='f' khóa phụ
9 select * from sys.objects where type='p' thủ tục lưu trữ dữ liệu
10 create database name(tên mình đặt database) => trong đó có nhiều tham số để set cho nó câu lệnh tạo database
11 CREATE DATABASE ten_co_so_du_lieu; dùng để tạo database
12 DROP DATABASE ten_co_so_du_lieu; để xóa một cơ sở dữ liệu hiện có trong SQL
13 CREATE TABLE Tên_bảng dùng để tạo bảng
14 INSERT INTO SINHVIEN (MASV, TENSV, QUEQUAN)VALUES (1, N’NGUYỄN VĂN A’, 1) thêm các bản ghi vào bảng
15 SELECT * FROM table_name; sử dụng để chọn dữ liệu từ cơ sở dữ liệu Cú pháp truy vấn dữ liệu tổng quát
16 SELECT column_name(s) FROM table1 INNER JOIN table2 ON table1.column_name = table2.column_name; Từ khóa INNER JOIN chọn các bản ghi có giá trị giống nhau trong cả hai bảng.(kết hợp cả 2 bảng vào với nhau)
17 ROW_NUMBER() OVER (ORDER BY tên_để_đánh_số ) hàm dùng để đánh số thứ tự
18 ROLLBACK TRAN "UPDATEDATA" dùng để quay lại khi xóa nhầm hoặc chạy lệnh nào đó nhầm
19 SET SQL_SAFE_UPDATES = 0; nếu ko xóa được dùng lệnh này kèm theo (MySQL)
